הצ׳ק-ליסט המוצע

הכל נגזר, אף פעם לא נשמר (ADR-0045). כל שורה פותחת את הפעולה עצמה, לא רק מחליפה טאב.

בדיקות

🏨לינה · קיימת הזמנת מלון? → פותח את טופס יצירת הלינה (סוג=מלון) קיים
📅ימים ריקים · יום ללא אירועים? → בונה היום על היום הריק הראשון קיים
✈️טיסות · הלוך וחזור — מושלם רק עם טיסה ליעד (הלוך) וטיסה מהיעד (חזור); → פותח את טופס יצירת הטיסה (סוג=טיסה, כיוון חסר) קיים+
👥הקבוצה · יותר מנוסע אחד הצטרף? → הזמנה מההגדרות קיים
🛂מסמכים ודרכונים · לכל נוסע דרכון? מדד לפי-נוסע (זמין אחרי ADR-0058) חדש
🔑קודי אישור · נודג׳ שלמות ירד — אסף: מינורי מדי לשורה משלו.

הוחלט (אסף, 2026-07-18)

✓ בדיקה חדשה: מסמכים/דרכונים בלבד; קודי-אישור ירד.
✓ כל כפתור פותח את הטופס הספציפי (לינה→טופס לינה, טיסה→טופס טיסה).
✓ טיסות: מושלם רק עם הלוך + חזור (טיסה ליעד וטיסה מהיעד).
✓ המוכנות מייעצת בלבד, לא חוסמת את מעבר ה"צאו לדרך".
✓ בדיקות שהושלמו מכווצות לסיכום.
✓ מסמכים — מדד לפי-נוסע ("2 מתוך 5"), עם פירוט בלחיצה.
