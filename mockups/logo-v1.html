04

Construction — Direction A

What we’d lock before wiring it into favicon.svg, the PWA icons, and the Login/Join lockups.

Rules it keeps

  • Color budget (ADR-0028). Teal = place, amber = the live now. Nothing decorative; the two meanings combine on purpose.
  • One loud element. The amber core glows; the marker body stays calm — the same “the trip is speaking” hierarchy as the board.
  • RTL-safe. Radially symmetric, so it never needs mirroring in the Hebrew-first layout.
  • Works to 16px. One silhouette, one accent — legible as a favicon.
  • Dark-mode ready. The mark carries its own ground, so it survives the token remap unchanged.

Clear space & min size

  • Clear space = the amber core’s diameter on every side.
  • Min size 16px favicon · 20px in-app top bar.
  • Ships as SVG (inline on Login/Join for crispness) + PNG 192/512/maskable for the PWA.
