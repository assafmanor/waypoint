כללי העיצוב (למימוש)

  • אשכול = רצף אירועים שחופפים בזמן (interval-merge). אירוע בודד ללא חפיפה נשאר שורה רגילה. end === start (גב-אל-גב) אינו חפיפה.
  • טיול — שקט. סוגר דק ניטרלי + כותרת קטנה. מבנה ודיו בלבד; אמבר נשאר לזמן/עכשיו.
  • תכנון — חפיפה ≠ פער. פער = כפתור מקווקו אוורירי בין שורות (מוסיף). חפיפה = קבוצה כרוכה בסוגר סגול + "הזז" בכותרת, בלי אלמנט בין־שורות. שפות מכוונות שונות.
  • דקדוק קשיח/רך נשמר בתוך האשכול (קשיח = כרטיס מלא + 🔒 + קוד; רך = מקווקו).
  • לוח = גיבור אחד. "עכשיו" ראשי לפי: קשיח > מסתיים־ראשון > מתחיל־ראשון; השאר ב"ועוד N". פיצול־קבוצה (בלי גיבור) הוא וריאנט.
  • מודל נתונים: deriveNow יחזיר nowAll[] / nextAll[] (קבוצות בו-זמניות), נגזר מהשעון, לא נשמר (ADR-0018); עוזר משותף clusterByTime מזין גם את היום וגם את הלוח.
