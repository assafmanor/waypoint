# Waypoint

[![CI](https://github.com/assafmanor/waypoint/actions/workflows/ci.yml/badge.svg)](https://github.com/assafmanor/waypoint/actions/workflows/ci.yml)

> **Working codename.** "Waypoint" is a placeholder for the trip control-center app — rename freely. The Hebrew product name in the mockup is **מרכז שליטה לטיול**.

A travel companion for a small group of friends traveling abroad together.

## The idea

Most travel apps help you **plan** a trip before you leave. Waypoint is a **living visibility layer** for when you're already there — it answers _"what now / what next / what do I need in the next 30 minutes?"_, and makes it easy to deviate from the plan without anyone getting lost or missing something important.

A few things that shape how it works:

- **Now / Next, not a wall of itinerary.** The app surfaces what matters at this moment, not the whole trip at once.
- **Some plans are firm, some aren't.** A flight or a reservation is locked in; a "maybe we'll grab lunch there" isn't. Waypoint treats these differently instead of pretending everything is equally fixed.
- **Everyone sees the same trip, live.** Changes one person makes show up for the rest of the group right away — and still work if you lose signal.
